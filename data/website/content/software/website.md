This is a simple website I created just for fun! It's made with Hugo.

The site is hosted on a DigitalOcean VPS. It's running nginx inside Docker and using LetsEncrypt for a wildcard SSL cert.

The site is also used as SSL termination for various self hosted services on my LAN.
This allows normally insecure services on my LAN be secure and private.
It also allows my services to have short and consistent URL's, while still supporting LAN routing when at home.

[Check out the code on GitHub](https://github.com/Repkap11/Website).
