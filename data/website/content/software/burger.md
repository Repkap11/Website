Burger is an Android app to find lunch buddies!

It shows you where other people are going for lunch and offers to send push notifications telling them "I'll Drive". The backend uses Google's firebase to synchronize data between users, send push notifications, and manage user accounts. 

[Check out the code on GitHub](https://github.com/Repkap11/Burger).
